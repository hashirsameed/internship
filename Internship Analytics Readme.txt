# Internship Analytics Project

This project provides a metrics-based system to evaluate and track intern performance using Python and pandas.

## Features

- Calculates key performance indicators (KPIs) such as:
  - Average task completion time
  - Average project quality rating
  - Average mentor feedback score
- Generates monthly performance reports in Excel format.
- Visualizes intern performance trends over time.

## How to Use

1. Ensure you have Python 3.x installed.
2. Install dependencies:
3. Run the script:
4. The script will generate sample data files (`tasks.csv`, `projects.csv`, `mentor_feedback.csv`), compute metrics, save a report (`intern_performance_monthly_report.xlsx`), and display a performance plot.
## Customize
- Replace sample data creation with your actual datasets.
- Modify KPIs and visualization as needed.
